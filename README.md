# Vaishnav V's Portfolio Website

A modern, responsive portfolio website showcasing my skills, education, work experience, projects, and certifications.

## Features

- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Dark/Light Theme**: Toggle between dark and light themes with persistent preference
- **Modern UI**: Glassmorphism effects, gradient backgrounds, and smooth animations
- **Smooth Navigation**: Fixed navbar with smooth scrolling and hover effects

## Pages

- **Home**: Introduction and overview with profile photo
- **Education**: Academic background and qualifications
- **Work Experience**: Professional experience and career history
- **College Projects**: Showcase of academic and personal projects
- **Skills**: Technical skills and proficiency levels
- **Certifications**: Professional certifications and courses completed
- **Contacts**: Contact information and contact form

## Technologies Used

- HTML5
- CSS3 (Custom styling with modern design patterns)
- JavaScript (Theme toggle functionality)
- Bootstrap 5.0.2 (Responsive framework)
- Google Fonts (Noto Sans, Playfair Display, Libre Baskerville)

## Theme Features

The website includes a dark/light theme toggle that:
- Persists user preference using localStorage
- Automatically applies saved theme on page load
- Provides smooth transitions between themes
- Ensures all text is readable in both themes

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. Navigate through the portfolio sections
